# Installation & Setup

## Download Telegram

You can download Telegram for various platforms:

### Mobile Devices
- [iOS App Store](https://apps.apple.com/app/telegram-messenger/id686449807)
- [Android Google Play](https://play.google.com/store/apps/details?id=org.telegram.messenger)
- [Android APK](https://telegram.org/android)

### Desktop
- [Windows](https://desktop.telegram.org/windows)
- [macOS](https://macos.telegram.org/)
- [Linux](https://desktop.telegram.org/linux)

### Web Version
- [Telegram Web](https://web.telegram.org/)

## Setting Up Your Account

1. **Download and Install**
   - Choose the appropriate version for your device
   - Install the application

2. **Create Account**
   - Open Telegram
   - Enter your phone number
   - Verify with the code sent via SMS
   - Set up your profile (optional):
     - Add a profile picture
     - Choose a username
     - Write a bio

3. **Security Settings**
   - Set up Two-Step Verification
   - Review Privacy Settings
   - Configure notification preferences

## Next Steps

After installation, you might want to:
- Add contacts
- Join groups or channels
- Start your first chat
- Explore Telegram features
