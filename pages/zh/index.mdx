# 如何使用 Telegram

欢迎来到 Telegram 使用指南！本指南将帮助你了解和掌握 Telegram 的各项功能。

## 什么是 Telegram？

Telegram 是一款注重速度、安全和隐私的云端即时通讯应用。它提供以下特色功能：

- 端对端加密聊天
- 大型群组对话
- 广播频道
- 最大 2GB 的文件分享
- 跨平台同步
